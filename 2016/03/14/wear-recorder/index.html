Record Audio on Wearable Devices

Posted on 2016-03-14

Wearable devices have great potential in our everyday life. For example, you can simply tap the watch screen or speak a command to wake up the watch and start recording, and then transmit the recorded clip to the phone for storage. This article will put down the procedures of developing a watch recorder.

Create a Recorder

Recording on wearable devices is basically the same as handheld devices, except that you cannot use MediaRecorder APIs for wearable devices. Instead, you must use AudioRecord to record voice. The implementation of AudioRecord is completely same as handheld devices. You can see my article for reference, Recording with AudioRecord APIs in Android

Transmit Data in Bytes

Instead of saving data stream to a File while recording in handheld devices, you should store all raw data together and then send it to your phone.


With a ByteArrayOutputStream(), you could store data bytes while recording. Remember to call it in a background thread after recording started.

Send Message to Handheld devices

With Wearable.MessageApi, you could send bytes data to other devices easily. You only need to specify a path for message and retrieve an available device id.
