然而，实际操作中发现，这种方法明显有一些延迟（因为项目中要加载的图片已经在内存中，所以不是图片加载的问题），导致用户体验不太好，屏幕要黑屏一小段时间。所以试着用自己的代码实现了缩放，效果比上面的实现要好上很多：

            public void onSuccess() {
                Drawable drawable = imageView.getDrawable();
                drawable = zoomDrawable(drawable, SCREEN_WIDTH);
                imageView.setImageDrawable(drawable);
            }

            @Override
            public void onError() {

            }

安卓手势学习笔记（三） 多点触控

Posted on 2015-09-12

跟踪多个触点

当多个手指同时触碰屏幕时，系统产生如下的事件：

  • ACTION_DOWN –第一个触点。它启动了手势，在MotionEvent中该触点的数据索引是0.
  • ACTION_POINTER_DOWN –第一个触点之后触碰屏幕的其他触点。其索引通过getActionIndex()获取。
  • ACTION_MOVE –已经按下的手势发生变化。
  • ACTION_POINTER_UP –非最初手指抬起时发送。
  • ACTION_UP –最后一个手指离开屏幕时发送。
