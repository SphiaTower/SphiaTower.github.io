安卓动画学习笔记（四） 属性动画（续）

Posted on 2015-09-13

用AnimatorSet编排多个动画

很多时候，播放动画的时机取决于其他动画的开始和结束。安卓系统可以将多个动画捆绑在一个AnimatorSet中，以指定同时播放动画，按顺序播放动画或者延迟一定时间后播放。AnimatorSet对象也可以互相嵌套。
下面的例子取自一个弹球游戏，播放下面的顺序播放Animator对象：

  1. Plays bounceAnim
  2. Plays squashAnim1, squashAnim2, stretchAnim1, and stretchAnim2 at the same time.
  3. Plays bounceBackAnim.

  4. Plays fadeAnim.

    AnimatorSet bouncer = new AnimatorSet();
    bouncer.play(bounceAnim).before(squashAnim1);
    bouncer.play(squashAnim1).with(squashAnim2);
    bouncer.play(squashAnim1).with(stretchAnim1);
    bouncer.play(squashAnim1).with(stretchAnim2);
    bouncer.play(bounceBackAnim).after(stretchAnim2);
    ValueAnimator fadeAnim = ObjectAnimator.ofFloat(newBall, “alpha”, 1f, 0f);
    fadeAnim.setDuration(250);
    AnimatorSet animatorSet = new AnimatorSet();
    animatorSet.play(bouncer).before(fadeAnim);
    animatorSet.start();

动画侦听器

在动画过程中，可以侦听一些重要事件。

  • Animator.AnimatorListener

    • onAnimationStart()
    • onAnimationEnd()
    • onAnimationRepeat()
    • onAnimationCancel() 动画取消时也将调用onAnimationEnd()

  • ValueAnimator.AnimatorUpdateListener

    • onAnimationUpdate() 在动画的每一帧都会调用。在动画过程中，侦听这个时间可以使用ValueAnimator计算得出的值。通过调用传入的ValueAnimator的getAnimatedValue()可以获得当前的值。如果使用ValueAnimator必须要实现这个侦听器。 有时需要调用invalidate()方法，详见上一篇文章。

ViewGroup的布局变化动画

属性动画系统可以为ViewGroup布局变化提供动画，也为View提供了一种简单的动画方式。
可以使用LayoutTransition类来为布局变化提供动画。ViewGroup添加或移除View或者设置属性为VISIBLE``INVISIBLE``GONE时可以添加出现和消失的动画。ViewGroup中剩下的View也会通过动画移动到新的位置。可以通过调用setAnimator()方法来为LayoutTransition提供Animator，并选择下面几个常数之一：

  • APPEARING 新的item出现时执行
  • CHANGE_APPEARING item因为新item添加而改变位置时执行
  • DISAPPEARING item消失时执行
  • CHANGE_DISAPPEARING item因为其他item消失而改变位置时执行

可以为这四种类型的时间提供自定义的动画，也可以使用默认的动画
layout_animations_by_default.xml。
使用默认动画时需要将ViewGroup的android:animateLayoutchanges属性设置为true。

<LinearLayout
    android:orientation="vertical"
    android:layout_width="wrap_content"
    android:layout_height="match_parent"
    android:id="@+id/verticalContainer"
    android:animateLayoutChanges="true" />

这样就可以自动处理ViewGroup中添加删除View的动画。

View的动画

属性动画系统可以为View对象提供streamlined动画，并提供一些视图动画不具有的优势。视图动画仅仅改变了View绘制的位置，由每个View的容器进行处理，而View对象本身没有变化。导致一些行为仍然留在原来的位置，即使画在别的位置上。安卓3.0以后，添加了新的属性和相应的getter和setter方法来弥补这个不足。
属性动画系统可以通过改变实际的属性来操纵View的动画。而且，View可以在属性改变之后自动调用invalidate()方法来刷新屏幕。View类中支持属性动画的新的属性有：

  • translationX和translationY：这些属性控制View摆放位置的偏移增量，相对其布局容器设定的左上角坐标。
  • rotation,rotationX和rotationY：这些属性控制围绕中轴点的2D和3D旋转。
  • scaleX和scaleY：这些属性控制View围绕中轴点的2D缩放大小。
  • pivotX和pivotY：这些属性控制中轴点的位置。旋转和缩放都围绕着中轴点进行。默认情况下，中轴点位于对象的中心位置。
  • x和y：这些属性描述View在容器中的最终位置，是left和top值与translationX和translationY值得总和。
  • alpha：代表View的透明度。默认值为1（不透明），0代表完全透明。

要改变View的属性，比如颜色或者旋转，只需要创建一个属性animator，并制定想要添加动画的View属性，比如：

ObjectAnimator.ofFloat(myView, "rotation", 0f, 360f);

使用ViewPropertyAnimator

ViewPropertyAnimator利用其中的Animator对象，提供了简单的方法来为View的几个属性并行地实现动画。它与ObjectAnimator的表现类似，修改实际的属性值，但在同时改变多个属性的时候效率更高。而且它的代码也更加紧凑易读。下面的片段体现了几种实现方法的区别：

多个ObjectAnimator对象

ObjectAnimator animX = ObjectAnimator.ofFloat(myView, "x", 50f);
ObjectAnimator animY = ObjectAnimator.ofFloat(myView, "y", 100f);
AnimatorSet animSetXY = new AnimatorSet();
animSetXY.playTogether(animX, animY);
animSetXY.start();

单个ObjectAnimator

PropertyValuesHolder pvhX = PropertyValuesHolder.ofFloat("x", 50f);
PropertyValuesHolder pvhY = PropertyValuesHolder.ofFloat("y", 100f);
ObjectAnimator.ofPropertyValuesHolder(myView, pvhX, pvyY).start();

ViewPropertyAnimator

myView.animate().x(50f).y(100f);

在XML中声明动画

属性动画系统支持使用XML来声明动画。通过在XML中定义动画，可以简单地在多个activity中重用动画，编辑动画顺序也更加容易。
为了与传统的视图动画相区分，从安卓3.1开始，应该将属性动画的XML文件存储在res/animator目录（而非res/anim/）。使用animator目录名并非强制的，但在使用一些插件时会更加方便。
下面的属性动画类通过相应的标签支持XML声明：

  • ValueAnimator
  • ObjectAnimator
  • AnimatorSet

下面的例子按次序播放两套对象动画，第一套嵌套着两个同时播放的对象动画：

<set android:ordering="sequentially">
    <set>
        <objectAnimator
            android:propertyName="x"
            android:duration="500"
            android:valueTo="400"
            android:valueType="intType"/>
        <objectAnimator
            android:propertyName="y"
            android:duration="500"
            android:valueTo="300"
            android:valueType="intType"/>
    </set>
    <objectAnimator
        android:propertyName="alpha"
        android:duration="500"
        android:valueTo="1f"/>
</set>

为了运行这个动画，必将在代码中把XML资源inflate给一个AnimatorSet对象，在启动动画集合之前，必须为所有对象设置目标对象。调用setTarget()可以方便地为AnimatorSet中的所有动画设置同一个目标对象。

AnimatorSet set = (AnimatorSet) AnimatorInflater.loadAnimator(myContext,
    R.anim.property_animator);
set.setTarget(myObject);
set.start();
