安卓动画学习笔记（二） 自定义ViewPager动画

Posted on 2015-09-13

使用PageTransformer自定义动画

可以实现ViewPager.PageTransformer接口来实现自定义的滑动动画。这个接口只有一个方法transformPage()。在屏幕切换的每一个点上，这个方法为当前可见的页面和相邻的页面各调用一次。比如，当前显示第三页，用户滑动到第四页，transformPage()在手势的每一步都为第二，三，四页所调用。
实现transformPage()时，可以创建自定义的滑动动画。根据页面在屏幕上的位置（通过方法的position参数获取），决定哪一个页面需要被变形。

position是一个随用户滑动页面而改变的动态参数，指示了页面相对屏幕中心的位置。页面充满屏幕时，其值为0。页面刚刚从屏幕右侧切出时，值为1。如果用户滑动至两个页面的中间位置，第一个页面为-0.5,第二个页面为0.5.根据页面的位置，可以通过setAlpha(), setTranslationX()或者setScaleY()等方法来设置页面属性，创建自定义动画。
调用setPageTransformer()来应用自定义动画。代码如下：

ViewPager mPager = (ViewPager) findViewById(R.id.pager);
...
mPager.setPageTransformer(true, new ZoomOutPageTransformer());

缩小切换动画

在切换页面过程中，缩小并淡出页面。当一个页面接近中心时在恢复到正常的尺寸并淡入。

public class ZoomOutPageTransformer implements ViewPager.PageTransformer {
    private static final float MIN_SCALE = 0.85f;
    private static final float MIN_ALPHA = 0.5f;

    public void transformPage(View view, float position) {
        int pageWidth = view.getWidth();
        int pageHeight = view.getHeight();

        if (position < -1) { // [-Infinity,-1)
            // This page is way off-screen to the left.
            view.setAlpha(0);

        } else if (position <= 1) { // [-1,1]
            // Modify the default slide transition to shrink the page as well
            float scaleFactor = Math.max(MIN_SCALE, 1 - Math.abs(position));
            float vertMargin = pageHeight * (1 - scaleFactor) / 2;
            float horzMargin = pageWidth * (1 - scaleFactor) / 2;
            if (position < 0) {
                view.setTranslationX(horzMargin - vertMargin / 2);
            } else {
                view.setTranslationX(-horzMargin + vertMargin / 2);
            }

            // Scale the page down (between MIN_SCALE and 1)
            view.setScaleX(scaleFactor);
            view.setScaleY(scaleFactor);

            // Fade the page relative to its size.
            view.setAlpha(MIN_ALPHA +
                    (scaleFactor - MIN_SCALE) /
                    (1 - MIN_SCALE) * (1 - MIN_ALPHA));

        } else { // (1,+Infinity]
            // This page is way off-screen to the right.
            view.setAlpha(0);
        }
    }
}

深度切换动画

深度切换动画将页面淡出，并线性缩小。在动画过程中，默认动画（屏幕滑动）仍然生效，所以必须用一个反方向的X translation来抵消屏幕滑动，如：

view.setTranslationX(-1 * view.getWidth() * position);

下面是完整的代码：

public class DepthPageTransformer implements ViewPager.PageTransformer {
    private static final float MIN_SCALE = 0.75f;

    public void transformPage(View view, float position) {
        int pageWidth = view.getWidth();

        if (position < -1) { // [-Infinity,-1)
            // This page is way off-screen to the left.
            view.setAlpha(0);

        } else if (position <= 0) { // [-1,0]
            // Use the default slide transition when moving to the left page
            view.setAlpha(1);
            view.setTranslationX(0);
            view.setScaleX(1);
            view.setScaleY(1);

        } else if (position <= 1) { // (0,1]
            // Fade the page out.
            view.setAlpha(1 - position);

            // Counteract the default slide transition
            view.setTranslationX(pageWidth * -position);

            // Scale the page down (between MIN_SCALE and 1)
            float scaleFactor = MIN_SCALE
                    + (1 - MIN_SCALE) * (1 - Math.abs(position));
            view.setScaleX(scaleFactor);
            view.setScaleY(scaleFactor);

        } else { // (1,+Infinity]
            // This page is way off-screen to the right.
            view.setAlpha(0);
        }
    }
}
