使用SAXParser解析XML

Posted on 2015-10-03

SAXParser是一个用来解析XML文件的工具，相比起使用正则表达式，效率更高，可操作性更强。使用起来有一些问题需要注意。

创建解析器

首先通过工厂方法拿到SAXParser的实例，接下来用InputSource类来包裹XML文件，然后再调用parse解析时，传入一个DefaultHandler的子类，来告诉parser解析的方式，并存储解析的结果。

SAXParser saxParser = SAXParserFactory.newInstance().newSAXParser();
InputSource inputSource = new InputSource();
inputSource.setEncoding("UTF-8");
inputSource.setCharacterStream(new StringReader(xml));
XmlHandler defaultHandler = new XmlHandler();

saxParser.parse(inputSource, defaultHandler);

DefaultHandler

SAXParser的解析过程很有意思。它在解析中不断生成各种事件：startDocument(), startElement(), characters(), endElement(), endDocument()，分别表示开始解析文档、开始解析标签、取出中间的字符、解析关闭标签、结束解析文档。开发者要做的就是，利用这些事件的回调方法，来取回自己想要的结果。因为这些事件方法之间在时间上互不关联，所以自然需要开发者自己在类中存储状态信息。

具体实现就是添加一些布尔field，比如isTitleHit，isLinkHit等等。解析标签内的内容时，在startElement()中创建一个StringBuffer，在characters()中append（ch,int,int），最后在endElement()中完成String的创建，根据布尔field将之存储起来，作为一次扫描的结果。

注意，在一个标签内可能会调用多次characters()，所以一定要在endElement()中结束一次标签的扫描。

注意，XML中扫描到&amp;字符会断开，但不可以替换为&，这样会损坏XML中间，解决方法如上一条所述。

下面是一个解析RSS的例子：

class XmlHandler extends DefaultHandler {
    private boolean titleHit = false;
    private boolean linkHit = false;
    private boolean itemHit = false;
    private List<String> titles = new ArrayList<>();
    private List<String> links = new ArrayList<>();
    private StringBuffer mStringBuffer;

    @Override
    public void startElement(String uri, String localName, String qName, Attributes attributes) throws SAXException {
        super.startElement(uri, localName, qName, attributes);
        if (itemHit) {
            switch (localName) {
                case "title":
                    titleHit = true;
                    break;
                case "link":
                    linkHit = true;
                    break;
                default:
                    return;
            }
        }
        if (localName.equals("item")) {
            itemHit = true;
        }
        mStringBuffer = new StringBuffer();
    }

    @Override
    public void endElement(String uri, String localName, String qName) throws SAXException {
        super.endElement(uri, localName, qName);
        if (itemHit) {
            if (localName.equals("title")) {
                titleHit = false;
                titles.add(new String(mStringBuffer));
            } else if (localName.equals("link")) {
                linkHit = false;
                links.add(new String(mStringBuffer));
            }
        }
        if (localName.equals("item")) {
            itemHit = false;
        }
    }

    @Override
    public void characters(char[] ch, int start, int length) throws SAXException {
        super.characters(ch, start, length);
        if (itemHit) {
            if (titleHit || linkHit) {
                mStringBuffer.append(ch, start, length);
            }
        }
    }
}
